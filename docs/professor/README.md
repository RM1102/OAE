# OAE Professor Install Guide

## 1) Unzip the package
- Extract `OAE-Professor-Kit-*.zip` to any folder.

## 2) Run installer
- Right-click `install.command` and choose **Open**.
- If macOS warns, confirm **Open**.

## 3) Finish in-app setup
- OAE launches a setup assistant.
- Complete these 3 steps in order:
  - Install bundled Whisper model files
  - Install/start Ollama
  - Pull Ollama model

## 4) Grant permissions
- Microphone: required for Dictate/Capture.
- Accessibility: required for hotkeys/autopaste features.

## 5) Start using OAE
- Open Dictate for live transcription.
- Open Post Process after Ollama step is ready.

## Notes
- This package is prepared to avoid first-run model download delays.
- If Ollama install fails, use the setup assistant retry buttons.
